구글 검색 결과에 사이트가 어떻게 노출되는지 확인하고 관리하는 구글 공식 도구입니다.                                                                                                                                  
  주요 기능:                                                                                                          
  - 색인 현황 — 구글이 내 사이트 페이지를 얼마나 크롤링/색인했는지                                                                                                                                                     
  - 검색 성능 — 어떤 키워드로 노출됐는지, 클릭수/노출수/순위                                                                                                                                                         
  - 크롤링 오류 — 봇이 접근 못한 페이지 확인
  - 사이트맵 제출 — sitemap.xml 등록해서 색인 요청
  - Core Web Vitals — 페이지 로딩 성능 리포트

  transfermap 입장에서 왜 필요한가:

  애드센스 수익은 방문자 수에 비례 → 구글 검색 유입이 핵심 → Search Console로 SEO 상태를 파악해야 개선 가능

  SEO (Search Engine Optimization) — 검색엔진 최적화.                                                                                                                                                                  
                                                                                                                                                                                                                     
  Google 같은 검색엔진이 내 사이트를 잘 이해하고, 관련 검색어에서 상위에 노출되도록 하는 작업.                                                                                                                         
   
  핵심 요소                                                                                                                                                                                                            
                                                                                                                                                                                                                     
  기술적 SEO
  - 페이지 로딩 속도
  - 모바일 대응 (반응형)
  - <meta> 태그, <title> 태그
  - robots.txt, sitemap.xml

  콘텐츠 SEO
  - 검색어(키워드)가 페이지 내용에 포함
  - 구조화된 HTML (<h1>, <h2> 등 시맨틱 태그)

  외부 SEO
  - 다른 사이트에서 내 사이트로 링크 (백링크)

  Transfer 프로젝트 기준

  어제 파일에서 완료한 항목들(Google Search Console이 뭔지 파악, SEO 이점 정리)이 이 맥락임.
  Search Console은 Google이 내 사이트를 어떻게 인식하는지 모니터링하는 도구고, 도메인 소유권 인증이 선행되어야 함
