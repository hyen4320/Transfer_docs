### 클럽 이름 매칭 문제 (Arsenal vs Arsenal FC)

**문제:** Claude API 파싱 결과가 "Arsenal"인데 DB는 "Arsenal FC" → 매칭 실패

**결론: Alias 테이블 방식 채택 (향후 확장성 최우선)**

```sql
CREATE TABLE club_aliases (
    id         BIGSERIAL PRIMARY KEY,
    club_id    BIGINT NOT NULL REFERENCES clubs(id),
    alias      VARCHAR(100) NOT NULL,
    lang       VARCHAR(10) DEFAULT 'en',  -- 다국어 대비
    source     VARCHAR(50),               -- 'manual' | 'transfermarkt' | 'wikipedia'
    UNIQUE (alias)
);
CREATE INDEX idx_club_aliases_alias ON club_aliases (lower(alias));
```

**왜 alias 테이블인가:**
- 리그 추가 시 → alias CSV만 추가
- 다국어 소스 ("바이에른 뮌헨" 등) → lang 컬럼으로 처리
- 나중에 `player_aliases`도 같은 패턴 재사용
- 사용자 검색 자동완성에도 그대로 활용 가능

**초기 시딩 전략:** crawl.py 실행 시 클럽명 변형 함께 수집 or Claude API 배치로 seed 생성
