## 인터랙션 컨셉

### 국가 진입 — Zoom-In (빨려들어가기)

별도의 Countries 페이지로 이동하는 대신, 메인 맵에서 국가를 직접 클릭하면 해당 국가로 빨려들어가는 방식으로 진입한다.

선수의 이적 상황을 비행기 관제화면으로 표현
**동작 순서:**
1. 5개 리그 국가(England / Spain / Germany / Italy / France)에 hover 시 글로우 효과로 클릭 가능함을 표시
2. 클릭 시 클릭한 국가의 중심점을 기준으로 맵 전체가 `scale(10)`까지 빠르게 줌인
3. 줌인되는 동안 리그 배지, 클럽 마커, 탑바 등 UI가 페이드아웃
4. 0.7초 애니메이션 후 해당 국가의 상세 맵 페이지로 전환

**의도:**
- 탐색 구조를 메뉴 이동이 아닌 공간 이동처럼 느끼게 한다
- 사용자가 지도 위에서 직접 세계로 "들어가는" 몰입감 제공
- 뒤로 가기 시에는 진입 경로에 따라 back 버튼 레이블이 달라짐 (← Map / ← Countries)
