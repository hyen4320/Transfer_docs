## 페이지별 레이아웃

### `/` 메인 (지도)

- **목적**: 서비스의 핵심. 유럽 지형 위에 리그/구단을 시각화
- **배경**: 블랙 계열 다크 지도 (Mapbox dark 스타일 참고)
- **주요 구성요소**:
  - D3.js Mercator 유럽 지도 (전체 화면)
  - 5대 리그 로고 마커 — 클릭 시 해당 리그 뉴스 패널 열기
  - 구단 위치 마커 (`CLUB.latitude / longitude`) — 클릭 시 구단 상세 패널
  - 이적 경로 선 (from_club → to_club, 애니메이션 화살표)
  - 좌상단: TransferMap 로고/타이틀
  - 우상단: 기자 랭킹 페이지 이동 버튼
  - 좌하단: 리그 필터 토글 (5대 리그 아이콘)
- **인터랙션**:
  - 리그별 국가 영역 hover → fill 변경
  - 구단 마커 클러스터링 (같은 도시 밀집 구단)
  - 줌 / 패닝
  - 이적 경로 애니메이션
- **API**: `GET /api/leagues/{id}/clubs`, `GET /api/news`

---

### `/` 이적 뉴스 피드 패널 (우측 슬라이드)

- **목적**: 지도를 유지한 채 최신 이적 뉴스를 확인
- **위치**: 우측 슬라이드 패널 (지도 위 오버레이, 너비 약 380px)
- **주요 구성요소**:
  - 시즌 셀렉터 — 변경 시 뉴스 피드 + 지도(구단·이적 경로) 동시 갱신
  - 뉴스 카드 리스트 (무한 스크롤, page size 30)
    - 현재 시즌: `publishedAt desc` 정렬
    - 과거 시즌: `feeEur desc` 정렬 (이적료 없는 항목 제외)
    - 선수명 클릭 → 좌측 선수 패널 열기 + 해당 국가로 지도 이동
    - 카드 클릭 → 선수 상세 페이지 이동
    - 상태 배지: `RUMOUR` `CONFIRMED` `DENIED` `LOAN`
    - 이적료 / 보도 시각 / 기자명
  - 지도 이적 경로 hover → 해당 뉴스 카드로 피드 자동 스크롤 (광고 위 120px 오프셋)
- **API**: `GET /api/news?season&page&size&sort&minFeeEur`

---

### `/journalists` 기자 랭킹

- **목적**: 이적 시장 주요 기자 신뢰도 랭킹 제공
- **주요 구성요소**:
  - 랭킹 테이블 (공신력 점수 순)
    - 순위 / X 핸들 / 이름 / 팔로워 수 / 공신력 점수
  - 점수 구성 시각화 바 (speed 0.3 / accuracy 0.5 / impact 0.2)
  - 기자 행 클릭 → `/journalists/:id` 이동
- **API**: `GET /api/journalists`

---

### `/journalists/:id` 기자 상세

- **목적**: 특정 기자의 프로필과 보도 이력 확인
- **주요 구성요소**:
  - 프로필 헤더: X 프로필 이미지, 이름, 핸들, 팔로워 수, 공신력 점수
  - 점수 세부 차트 (speed / accuracy / impact)
  - 보도 뉴스 카드 리스트 (최신순)
- **API**: `GET /api/journalists/{id}`, `GET /api/journalists/{id}/news`

---

### `/players/:id` 선수 상세

- **목적**: 특정 선수의 프로필과 이적 히스토리 확인
- **주요 구성요소**:
  - 프로필 헤더: 사진, 이름, 국적, 포지션, 현 소속팀, 계약 만료일
  - 이적 히스토리 타임라인 (연도별 from → to 시각화)
  - 관련 뉴스 카드 목록
- **API**: `GET /api/players/{id}`, `GET /api/players/{id}/transfers`

---

### `/` 선수 패널 (좌측 슬라이드) — 신규

- **목적**: 뉴스 피드에서 선수명 클릭 시 지도를 유지한 채 이적 이력 확인
- **위치**: 좌측 슬라이드 패널 (너비 460px, z-60), 조건 검색 패널과 동일 슬롯
- **트리거**: 뉴스 카드 선수명 클릭 → `fetchPlayersSearch(name, 1)`로 ID 조회 → 패널 표시 + 지도 해당 국가 이동
- **주요 구성요소**:
  - 프로필 카드: 국적, 포지션, 현 소속팀, 리그, 계약 상태
  - 이적 히스토리 타임라인
  - 관련 뉴스 카드 목록
- **API**: `GET /api/players/{id}`, `GET /api/players/{id}/transfers`, `GET /api/players/search?name`

---

### `/` 구단 상세 패널 (우측 슬라이드)

- **목적**: 지도의 구단 마커 클릭 시 지도를 유지한 채 구단 정보 표시
- **위치**: 우측 슬라이드 패널 (뉴스 피드 패널과 동일 슬롯, 상호 전환)
- **주요 구성요소**:
  - 구단 헤더: 로고, 이름, 도시, 리그, 경기장
  - Incoming / Outgoing 탭 전환
  - 이적 뉴스 카드 리스트
- **특이사항**: 뉴스 피드 패널과 같은 패널 영역 공유 — 마커 클릭 시 구단 패널로 전환, 닫기 시 뉴스 피드로 복귀
- **API**: `GET /api/clubs/{id}`, `GET /api/clubs/{id}/transfers`
