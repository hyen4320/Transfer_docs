# 이적시장 뉴스만 골라내기

X(트위터)에서 수집한 기자 게시글 중 **이적 관련 게시글만 필터링**하고, 구조화된 데이터로 변환하는 파이프라인 설계.

---

## 핵심 문제

기자들은 이적 뉴스 외에도 경기 리뷰, 일상, 의견글 등 다양한 게시글을 올린다.
→ 수집된 Post 중 실제 이적 관련 게시글만 골라내야 한다.

---

## 방법 비교

### 방법 1. 규칙 기반 키워드 매칭 (MVP 추천)

DB의 Player/Club 이름 목록 + 이적 키워드 사전으로 1차 필터링.

**이적 상태 키워드 사전:**

| Status       | 키워드 예시                                              |
|--------------|----------------------------------------------------------|
| `CONFIRMED`  | confirmed, done deal, official, signs, completed, sealed |
| `RUMOR`      | close to, in talks, discussing, interest, bid, offer     |
| `DENIED`     | denies, rejects, not leaving, stays, ruled out           |
| `LOAN`       | loan deal, on loan, loan move, loan fee                  |
| `INTEREST`   | interested, targeting, considering, monitoring           |

**필터 조건 (AND):**
1. 키워드 사전에서 1개 이상 매칭
2. DB Player 이름 OR Club 이름 매칭

**장점:** 빠름, 비용 없음  
**단점:** 신조어/축약어에 취약, Player·Club 목록 의존

---

### 방법 2. Claude API 구조화 추출 (본 구현 추천)

1차 필터를 통과한 게시글을 Claude API에 넘겨 JSON 추출.

**프롬프트 예시:**
```
다음 축구 기자의 트윗에서 이적 관련 정보를 추출하세요.
이적 뉴스가 아니면 null을 반환하세요.

트윗: "{post_text}"

출력 형식:
{
  "is_transfer_news": true/false,
  "players": ["이름"],
  "from_club": "클럽명 or null",
  "to_club": "클럽명 or null",
  "status": "INTEREST|RUMOR|CONFIRMED|DENIED|LOAN",
  "window": "SUMMER|WINTER|null"
}
```

**장점:** 노이즈 텍스트에 강함, 언어 무관, 학습 불필요  
**단점:** API 비용 발생 (배치 처리로 절감 가능)

---

### 방법 3. spaCy NER (중간)

영어 NER 모델로 인명(PERSON) / 조직(ORG) 추출 후 규칙 결합.
Python 스크립트로 구현, 결과를 BE에 전달.

**장점:** 비용 없음, 인명/조직 인식 정확도 양호  
**단점:** 축구 도메인 특화 모델 아님, 선수명 오인식 가능

---

## 추천 구현 전략

```
[수집된 Post]
     ↓
[1단계: 규칙 기반 1차 필터] ← 빠른 탈락 처리 (비용 절감)
     ↓ 통과
[2단계: Claude API 구조화 추출] ← 정확도 확보
     ↓
[TransferNews 저장]
```

- 1단계에서 명백히 이적 무관 게시글 탈락 → API 호출 최소화
- 2단계에서 구조화된 데이터 추출 → DB 저장 직결

## 더미 데이터 설계 (테스트용)

```java
// 이적 뉴스 O
"BREAKING: Kylian Mbappe to leave Real Madrid this summer. Done deal - confirmed."
"Arsenal are in advanced talks to sign Victor Osimhen. Loan move possible."
"OFFICIAL: Marcus Rashford joins Aston Villa on a permanent deal. Signed."

// 이적 뉴스 X (필터링 대상)
"What a match last night. City were incredible in the second half."
"My player of the season so far: Salah. No contest."
"Injuries piling up at Old Trafford. Concerning signs."
```
